
# rTechManager - Background Service Setup (Linux Mint)

1. SYSTEMD SERVICE CREATION
# Create a systemd service to run rTechManager in the background on startup
sudo tee /etc/systemd/system/rtechmanager.service <<EOF
[Unit]
Description=rTechManager Server Console
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=$(pwd)
ExecStart=/usr/bin/node server.js
Restart=always
Environment=NODE_ENV=production
Environment=API_KEY=${API_KEY}

[Install]
WantedBy=multi-user.target
EOF

2. ACTIVATE SERVICE
sudo systemctl daemon-reload
sudo systemctl enable rtechmanager
sudo systemctl start rtechmanager

3. CHECK STATUS
sudo systemctl status rtechmanager

4. VNC SETUP (If prompted)
# Ensure VNC password is set for the background service
vncpasswd

The app is now running in the background and will start automatically when your Linux Mint server boots.
Access at: http://your-ip:1783
