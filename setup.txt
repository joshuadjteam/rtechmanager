# rTechManager - Simple Setup for Linux Mint

1. INSTALL DEPENDENCIES (One-liner)
sudo apt update && sudo apt install -y git nodejs npm build-essential libpam0g-dev tightvncserver websockify novnc

2. DOWNLOAD & START (Copy-paste)
git clone https://github.com/joshuadjteam/rtechmanager.git
cd rtechmanager
npm install
npm run build
vncpasswd
sudo node server.js

3. FIREWALL (Optional but recommended)
sudo ufw allow 1783/tcp

Access: http://your-server-ip:1783
Login: Use your Linux system username and password.
