# rTechManager - Simple Deployment Guide

This guide describes how to deploy the rTechManager console on your Linux Mint server.

STEP 1: INSTALL CORE DEPENDENCIES
Run these commands to prepare your system for the dashboard and remote desktop:
sudo apt update
sudo apt install -y git curl build-essential python3-pip tightvncserver websockify libvirt-daemon-system libvirt-clients qemu-kvm iproute2 libpam0g-dev

STEP 2: INSTALL NODE.JS 22
rTechManager requires the latest stable Node environment:
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt install -y nodejs

STEP 3: CLONE THE REPOSITORY
Use GIT to pull the source code directly to your server:
git clone https://github.com/joshuadjteam/rtechmanager
cd rtechmanager

STEP 4: INSTALL PROJECT PACKAGES & BUILD
Prepare the frontend and install the backend modules:
npm install
npm run build

STEP 5: RUN THE SERVER
Launch the manager on port 1783:
node server.js

The console is now accessible at http://your-server-ip:1783
Log in using your existing Linux Mint username and password.
