# rTechManager v1.3 Update Guide

1. PULL LATEST CODE
cd /opt/rtechmanager
git pull origin main

2. REINSTALL DEPENDENCIES (New WS & FS modules)
npm install
npm run build

3. REGISTER CLI TOOL
# Make the rtech command global
sudo chmod +x /opt/rtechmanager/rtech
sudo ln -sf /opt/rtechmanager/rtech /usr/local/bin/rtech

4. RESTART
rtech stop
rtech start -b

Update complete. You can now use the 'rtech' command anywhere!
